【Seedream Tools 使用说明】 1. 填写 api_key 在右上角「配置文件」中填写你的 API Key。 2. 模型选择 V4 / V45 / V5 点击切换，生成更稳定。 3. 模式说明 - 文生图：只需要提示词 - 图生图：1张参考图 + 提示词 - 多图融合：2张以上参考图 - 组图生成：批量生成一组图 4. 参考图URL 一行一个，空行会自动忽略。 5. 生成结果 图片自动保存到桌面，可在配置中修改路径。 6. 常见问题 - 不支持的模型：检查 modelKey 是否正确 - 生成失败：检查 API_KEY、网络、额度 - 修改配置后建议重启应用

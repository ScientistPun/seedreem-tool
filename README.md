# Meow

> AI 图像生成桌面工具，支持文生图、图生图、多图融合、组图生成，内置提示词库、图片预览、自动 / 手动保存图片功能。

------

## 快速开始

```bash
# 克隆并安装
git clone <仓库地址>
cd seedreem-tool
npm install

# 启动（首次启动会自动创建配置文件）
npm start

# 在应用内点击右上角「配置」按钮，填入你的 API Key
```

------

## 功能说明

### 1. 配置说明

配置文件位于 `config.yml`，可通过界面右上角「配置」按钮打开配置弹窗进行设置。

#### 系统设置
| 配置项 | 说明 |
|--------|------|
| 调试模式 | 开启后显示详细日志信息 |
| 自动保存 | 开启后自动保存生成的图片到指定目录 |
| 保存地址 | 图片保存目录（自动保存时使用） |
| 图片格式 | 保存图片的格式，支持 PNG、JPEG |
| 保存前缀 | 图片文件名的前缀部分 |

#### 火山引擎（即梦）
| 配置项 | 说明 |
|--------|------|
| api_key | 火山引擎 API 密钥 |
| base_url | API 请求地址（默认：`https://ark.cn-beijing.volces.com/api/v3/images/generations`） |
| 即梦5.0 | 即梦 v5 模型 ID（默认：`doubao-seedream-5-0-260128`） |
| 即梦4.5 | 即梦 v4.5 模型 ID（默认：`doubao-seedream-4-5-251126`） |
| 即梦4 | 即梦 v4 模型 ID（默认：`doubao-seedream-4-0-250828`） |
| 水印 | 是否在生成的图片上添加水印 |

#### MiniMax
| 配置项 | 说明 |
|--------|------|
| api_key | MiniMax API 密钥 |
| base_url | API 请求地址（默认：`https://api.minimaxi.com/v1/image_generation`） |
| image-01 | image-01 模型 ID（默认：`image-01`） |
| image-01-live | image-01-live 模型 ID（默认：`image-01-live`） |


### 2. 图片保存

- 统一保存规则，自动保存与手动保存共用一套逻辑

- 文件名格式：`{前缀}{序号}.{后缀}`
- 默认前缀：`meow_YYMMDDHHMMSS_`（如 `meow_260318143045_0.png`）
- 自定义前缀支持日期占位符：`%yyyy`(四位年), `%yy`(两位年), `%mm`(月), `%dd`(日), `%HH`(时), `%MM`(分), `%SS`(秒)
  - 例：`meow_%yy%mm%dd_` → `meow_260318_0.png`

- 自动保存：开启配置并选择存储目录

- 手动保存：勾选图片 → 点击「保存选中图片」→ 选择目录

### 3. 提示词库

- 快速添加、删除常用提示词
- 点击卡片一键填入提示词输入框
- 本地持久化存储，重启不丢失

------

## 使用流程

1. 填写 `api_key`
2. 选择模型、生成模式
3. 填写提示词、参考图、尺寸等参数
4. 点击 🚀 开始生成
5. 在生成结果中预览 / 保存图片

------

## 常见问题

- 本应用根据火山引擎提供的接口而开发，详细信息可浏览官方文档
- 无法生成：检查 api_key、网络、日志
- 保存失败：检查目录权限、是否勾选图片
- 界面异常：重启程序即可

# Seedream Tools

AI 图像生成桌面工具，支持文生图、图生图、多图融合、组图生成，内置提示词库、图片预览、自动 / 手动保存图片功能。

------

## 配置文件说明（config.yml）

首次运行自动生成，所有核心配置统一管理

```yaml
# API 密钥，必填，用于服务鉴权
api_key: ""

# 开启调试模式
dev_mode: false

# 自动保存图片开关（默认）
auto_save: false
# 保存图片目录，自动保存开启时则有效
save_dir: 

# 模型接口地址
base_url: https://ark.cn-beijing.volces.com/api/v3/images/generations
# 默认留空使用程序内置接口，不懂请不要修改
models:
  v5: "doubao-seedream-5-0-260128"   # V5 模型接口
  v45: "doubao-seedream-4-5-251128"  # V45 模型接口
  v4: "doubao-seedream-4-0-250828"   # V4 模型接口

# 默认生成参数（程序自动生成，无需手动修改）
default:
  output_format: png      # 默认图片后缀，v5有效
  watermark: false        # 是否生成水印

```

------

## 功能说明

### 1. 图像生成功能

- 支持模型：**V4 / V45 / V5**

- 支持模式：

  - 文生图（text2img）
  - 图生图（img2img）
  - 多图融合（img2img_multi）
  - 组图生成（group）

  

- 可配置：尺寸、风格强度、参考图 URL、提示词

### 2. 图片保存功能

统一保存规则，自动保存与手动保存共用一套逻辑

- 文件名格式：

  ```
  seedream_YYMMDDHis_序号.png
  ```

  - 示例：`seedream_260317235959_0.png`

  

- 自动保存：`config.yml` 开启 `auto_save: true`

- 手动保存：勾选图片 → 点击「保存选中图片」→ 选择目录

### 3. 提示词库功能

- 快速添加、删除常用提示词
- 点击卡片一键填入提示词输入框
- 本地持久化存储，重启不丢失

### 4. 图片预览功能

- 点击图片打开大图预览
- 支持：关闭按钮、ESC 关闭、点击遮罩关闭
- 支持左右箭头切换上一张 / 下一张

### 5. 菜单功能

点击右上角 `⋯` 打开菜单

- ⚙️ 配置文件：快速编辑 config.yml
- 📖 说明文档：打开本 README（Markdown 渲染）
- 📝 查看日志：查看运行与报错日志
- ❌ 退出应用：关闭程序

### 6. 快捷键

- `F12`：打开 / 关闭开发者调试工具
- `ESC`：关闭所有弹窗与预览窗口

------

## 使用流程

1. 打开 `config.yml` 填写 `api_key`
2. 选择模型、生成模式
3. 填写提示词、参考图、尺寸等参数
4. 点击 🚀 开始生成
5. 在生成结果中预览 / 保存图片

------

## 常见问题

- 无法生成：检查 api_key、网络、日志
- 保存失败：检查目录权限、是否勾选图片
- 界面异常：重启程序即可

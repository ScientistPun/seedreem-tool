# Meow

> AI 图像生成桌面工具，支持文生图、图生图、多图融合、组图生成，内置提示词库、图片预览、自动 / 手动保存图片功能。

------

## 功能说明

### 1. 图像生成

- 支持模型：**V4 / V45 / V5**

- 支持模式：

  - 文生图（text2img）
  - 图生图（img2img）
  - 多图融合（img2img_multi）
  - 组图生成（group）
  

- 可配置：尺寸、风格强度、参考图 URL、提示词
  
- 分辨率


### 2. 图片保存

- 统一保存规则，自动保存与手动保存共用一套逻辑

- 文件名格式：`{前缀}{序号}.{后缀}`
- 默认前缀：`meow_YYMMDDHHMMSS_`（如 `meow_260318143045_0.png`）
- 自定义前缀支持日期占位符：`%yyyy`(四位年), `%yy`(两位年), `%mm`(月), `%dd`(日), `%HH`(时), `%MM`(分), `%SS`(秒)
  - 例：`meow_%yy%mm%dd_` → `meow_260318_0.png`

- 自动保存：开启配置并选择存储目录

- 手动保存：勾选图片 → 点击「保存选中图片」→ 选择目录

### 3. 提示词库

- 快速添加、删除常用提示词
- 点击卡片一键填入提示词输入框
- 本地持久化存储，重启不丢失

------

## 使用流程

1. 填写 `api_key`
2. 选择模型、生成模式
3. 填写提示词、参考图、尺寸等参数
4. 点击 🚀 开始生成
5. 在生成结果中预览 / 保存图片

------

## 常见问题

- 本应用根据火山引擎提供的接口而开发，详细信息可浏览官方文档
- 无法生成：检查 api_key、网络、日志
- 保存失败：检查目录权限、是否勾选图片
- 界面异常：重启程序即可
